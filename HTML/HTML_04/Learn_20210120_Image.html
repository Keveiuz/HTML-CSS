img.src

src: (Source) + Path(URL)

src: a propoty tells where the image from

img.alt

alt: (alternative) + Text

alt: Text in 'alt' will be shown when the image can not be find in 'src' path

img.title

title: (Reminder) Text

title: Text in 'title' will be shown when the cursor is hovering the image

img.width and img.height

size: Resizing the image to fit the window
