1. Annotation

(1)Syntax: <!-- Annotation -->

2. Special Character

(1) '&' in html is equivalent to '\' in C++

(2) < : &lt (less than)

(3) > : &gt (greater than)

(4) blank: &nbsp

(5) & : &amp

(6) add ';' to the end of all these Characters to activate them.
