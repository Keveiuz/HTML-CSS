List

1. Order List

Items in order list will start with ordered-numbers

  1. item_1
  2. item_2
  3. item_3

2. Unorder List

Items in unorder list will start with single Dot '.'

  • item_1
  • item_2
  • item_3

3. Self_Define List

Compared to formal list, DL has the element 'title' to illustrate the property of the list.

Item (Title)
item_1
item_2
item_3
